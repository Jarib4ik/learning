2. Возможно ли написать любую из этих функций так, чтобы она принимала 2 и больше параметров? Если да - предложите реализацию

Ответ: Да. возможно. 

function Sum(...arguments) {
    let result = 0;

    for (let i = 0; i < arguments.length; i++) {
        result += arguments[i];
    }
    return result;
}
